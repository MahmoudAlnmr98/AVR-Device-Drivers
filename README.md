
# AVR Device Drivers

### this repo contains device drivers for AVR atmega32 microcontroller using layered architecture

## the firmware in this repo is :
#### 1-DIO
#### 2-LCD
#### 3-Keypad
#### 4-switches
#### 5-interrupt
#### 6-Timers
#### 7-commumication protocols (UART, SPI, I2C)
#### 8-EEPROM
